# Empathy Map – UX Case Study

## Overview
This project focuses on understanding the needs, behaviors, frustrations, and goals of busy office professionals through an Empathy Map.

The objective was to research and analyze user pain points before designing a fitness application. By creating a detailed User Persona and Empathy Map, the project helps identify what users say, think, do, and feel in their daily lives.

## Problem Statement
Busy working professionals often struggle to maintain a healthy lifestyle due to time constraints, work pressure, and lack of motivation for regular exercise.

## Process
- Scenario Analysis
- User Persona Creation
- Pain Point Identification
- Empathy Mapping
- Solution Ideation
- Conclusion

## Tools Used
- Figma
- UX Research
- Canva

## Key Outcomes
- Better understanding of user behavior
- Identification of major pain points
- Creation of a structured Empathy Map
- User-centered solution recommendations
